# Using Behavioral AI to Fingerprint Large Language Models

This repository contains the data (`sequences_df.csv`) mentioned in the paper **Using Behavioral AI to Fingerprint Large Language Models**.

* **`IJCAI_ModelSequencesAnalysis.ipynb`**: Contains the visualizations of the sequences and some statistical tests.
* **`IJCAI_LSTMClassifierSequences.ipynb`**: Contains the trained model.

> **Note:** Please be wary of some minor differences in the results during the model training.
